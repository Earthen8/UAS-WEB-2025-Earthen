# SkyTravel - Flight Booking App ✈️
UAS Web Application Development - Semester Ganjil 2025

Aplikasi pencarian tiket pesawat realtime menggunakan Django dan Amadeus API.

# Fitur
- Search Flight (One Way / Round Trip)
- Realtime Availability (via Amadeus API)
- Responsive UI (Tailwind CSS - Dark Premium Theme)
- Booking Form Simulation

# Cara Menjalankan Project
1. Clone repository
2. Buat virtual environment & install dependencies:
  ```bash
  pip install django amadeus python-dotenv
